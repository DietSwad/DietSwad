# Diet Swad - Healthy Snacks from Rs299

A modern, responsive website for Diet Swad, featuring handcrafted healthy snacks made with pure, natural ingredients.

## Overview

Diet Swad offers 100% natural, guilt-free snacks including Power Bites, Royal Bites, Peanut-Sesame Delights, Butter Cookies, Roasted & Salted Cashews, and Peri-Peri Protein Mix. ## Thank You Page

The thank-you page (`thank-you.html`) features:
- **Dynamic Payment Integration**: Accepts payment amount via URL parameter (`?AMOUNT=xxx`)
- **Animated Success Indicator**: Beautiful checkmark animation with confetti, sparkles, and particle effects
- **UPI Payment Gateway**: Direct integration with UPI (UPI ID: 8017795017@superyes)
- **Payment Details Display**: Shows the exact payment amount dynamically
- **User-Friendly Interface**: Clean, responsive design with Diet Swad branding
- **Next Steps Information**: Clear guidance on what happens after payment
- **Fallback Support**: Manual payment option if UPI app doesn't open
- **Validation**: Error handling for invalid payment amounts

### Payment Flow
1. User submits order form with amount
2. Redirected to thank-you page with amount parameter
3. Page displays animated success message and payment amount
4. User clicks "Pay via UPI" button
5. Opens UPI app for payment
6. After payment, representative contacts via call/WhatsApp
